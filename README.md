# Healthcare Data Pipeline Demo

## Overview
This project demonstrates a simple healthcare data pipeline 
using Medallion Architecture (Bronze, Silver, Gold)
with orchestration using Azure Data Factory.

## Technologies Used 
- Python
- PySpark
- Azure Data Factory (ADF)
- Azure Databricks
- Medallion Architecture

## Architecture Flow
ADF (Data Ingestion)
Bronze (Raw Data Storage)
Silver (Data Cleaning & Transformation)
Gold (KPI Calculation & Reporting)
